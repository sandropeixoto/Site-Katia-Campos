# Kátia Campos - Psicologia Clínica

Landing page profissional desenvolvida para a psicóloga **Kátia Campos**, especializada em **Terapia Cognitivo-Comportamental (TCC)**. O projeto foca em oferecer uma experiência acolhedora, moderna e informativa para pacientes em busca de psicoterapia.

## 🌟 Diferenciais do Projeto

- **Design Acolhedor:** Paleta de cores baseada em tons terrosos e verdes suaves para transmitir calma.
- **Mobile-First:** Interface totalmente responsiva e otimizada para dispositivos móveis.
- **Performance:** Carregamento rápido utilizando padrões modernos de desenvolvimento estático.
- **SEO & Acessibilidade:** Estrutura semântica e conformidade com as melhores práticas de SEO.

## 🛠️ Tecnologias Utilizadas

- **Estrutura:** HTML5 Semântico.
- **Estilização:** CSS3 (Vanilla) com Design Tokens (Variáveis CSS).
- **Interatividade:** JavaScript (Vanilla) com animações de scroll (`IntersectionObserver`).
- **Ícones:** Phosphor Icons.
- **Tipografia:** Google Fonts (Outfit & Inter).

## 📂 Estrutura de Diretórios

```bash
├── .github/workflows/  # Configurações de CI/CD (GitHub Actions)
├── assets/             # Imagens e mídias locais
├── css/                # Arquivos de estilização (style.css)
├── js/                 # Lógica de interatividade (script.js)
├── index.html          # Ponto de entrada principal
├── .htmlhintrc         # Configuração de lint para HTML
├── .stylelintrc.json   # Configuração de lint para CSS
└── SUGGESTIONS.md      # Roadmap de melhorias futuras
```

## 🚀 CI/CD e Deploy

O projeto utiliza **GitHub Actions** para automação de processos:

1.  **Linting:** A cada push para a branch `main`, o código é validado via `htmlhint` e `stylelint`.
2.  **Deploy:** Após a validação, o site é publicado automaticamente no **GitHub Pages**.

## 💻 Como Executar Localmente

1.  Clone o repositório:
    ```bash
    git clone https://github.com/sandropeixoto/Site-Katia-Campos.git
    ```
2.  Abra o arquivo `index.html` em seu navegador ou utilize uma extensão de Live Server.
